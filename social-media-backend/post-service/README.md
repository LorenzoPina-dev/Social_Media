# post-service

Production-ready microservice for Social Media Platform.

## Features

- TypeScript with strict mode
- Express.js framework
- PostgreSQL database
- Redis caching
- Kafka messaging
- Comprehensive testing
- Docker support
- Kubernetes manifests
- Prometheus metrics
- Structured logging

## Getting Started

```bash
# Install dependencies
npm cinstall

# Run migrations
npm run migrate

# Start development server
npm run dev

# Run tests
npm run test

# Build for production
npm run build

# Start production server
npm start
```
